NEW ATTENDANCE FLOW
================================================================================

OLD FLOW:
  Click ▶ → WiFi Check → Timer Starts

NEW FLOW:
  Click ▶ → WiFi Check → FACE VERIFICATION → Timer Starts

STEPS:
  1. Student clicks triangle (▶) button
  2. Location permission check
  3. WiFi BSSID validation
  4. Face verification (NEW!)
     - Camera opens
     - Liveness detection
     - Capture 10 frames
     - Compare with stored embedding
     - Show result
  5. Timer starts (only if all pass)

================================================================================
SECURITY FEATURES
================================================================================

✅ Multi-layer validation (Location + WiFi + Face)
✅ Liveness detection (prevents photo/video attacks)
✅ High threshold (75% similarity required)
✅ Offline verification (privacy-focused)
✅ Encrypted storage (AES-256-GCM)
✅ 10-frame averaging (accuracy)

USER EXPERIENCE
================================================================================

1. Student logs in → Face data downloaded automatically
2. Student clicks ▶ button → WiFi check passes
3. Camera opens → "Position your face in frame"
4. Liveness detection → "Move your head", "Blink"
5. Capturing → "Frames: 1/10... 10/10"
6. Verification → "Face verified! Similarity: 87%"
7. Timer starts → Attendance tracking begins

If verification fails:
  ❌ "Face does not match. Similarity: 45%"
  Timer does NOT start
  Student can retry
